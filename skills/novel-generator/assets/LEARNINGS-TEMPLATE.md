# 项目状态文件模板

状态文件放在 `output/<小说名>/` 下。不要把具体小说状态写进 skill 目录。

```markdown
（上一章最后的事件、情绪和钩子）

## 下一章目标

- 关系变化：

## 待处理事项
- 未回收伏笔：
- 未解决误会：
- 必须出场角色：
```
