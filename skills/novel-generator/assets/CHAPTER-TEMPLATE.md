# 章节生成模板

用于生成女频投流付费短中篇小说章节。默认每章 2000-3000 字，必须落地爽点、必须章末卡点、必须产出金句，并且必须提供可切片名场面。只有用户明确要求短章时，才使用 1200-1600 字。

## 标准章节模板

```markdown
# 第XX章 [章节名]

> **本章概要**：
> **第一句话信息缺口**：
> **前 300 字任务**：（人设 + 冲突 + 谁受苦 + 谁施害 + 为什么关心）
> **本章爽点**（引用 PUNCHES.md 编号 + 一句话落点）：
> **本章锤点**：[受虐/施压 → 女主反应/小反抗 → 情绪升级]
> **情绪功能**：[心动/误会/拉扯/失控/决裂/和解/打脸/反转/追妻...]
> **关系进展**：
> **章末卡点**（引用 HOOKS.md 类型 + 具体落点）：
> **读者不看下一章会缺什么**：
> **可切片名场面**：
> **金句候选**（预期 1-2 句可截图传播）：

---

（正文 2000-3000 字；短章模式 1200-1600 字）

---

> **本章更新**：
> **下章目标**：
> **新增爽点 / 卡点 / 金句**：（需同步回 PUNCHES.md / HOOKS.md / GOLDEN-LINES.md）
```

## 章节检查清单

### 字数与结构

- [ ] 正文字数在 2000-3000 字；用户明确要求短章时才允许 1200-1600 字。
- [ ] 第一句话制造信息缺口，让读者产生"为什么"或"然后呢"。
- [ ] 前 300 字完成：人设、冲突、谁受苦、谁施害、读者为什么关心。
- [ ] 本章有明确场景，不是摘要堆砌。
- [ ] 开头承接上一章卡点或直接进入冲突。
- [ ] 结尾留下推进动力，章末卡点强度达标。

### 爽点与卡点

- [ ] 本章爽点落到具体动作 / 一句话 / 一个画面（不是抽象情绪）。
- [ ] 本章有清楚的连环锤：受虐/施压 → 女主反应/小反抗 → 情绪升级。
- [ ] 如果上一章偏虐，本章必须给小反抗、留证、退路或旁人替读者发声。
- [ ] 章末卡点强度匹配章节位置（前 6 章必须达到"极强"或"强"）。
- [ ] 章末写清"不看下一章会缺什么"，不能只留氛围。
- [ ] 至少产出 1 句可截图传播的金句候选。
- [ ] 至少有 1 个可切片名场面，前 6 章尤其必须具备。

### 女频核心

- [ ] 女主有主动选择、行动或清晰情绪反应。
- [ ] 男主三锚点至少触发了 1 个（视觉 / 语言 / 情绪）。
- [ ] 关系变化有因果，不跳步。
- [ ] 甜、虐、误会、暧昧都服务人物关系。
- [ ] 男主 / 关键角色不是纯工具人。
- [ ] 反派或阻力没有降智。
- [ ] 心理描写不超过本章 30%。

### 连贯性

- [ ] 与 `STATE.md` 的下一章目标一致。
- [ ] 角色状态与 `CHARACTERS.md` 一致。
- [ ] 没有违反 `STORY_BIBLE.md`。
- [ ] 已处理上一章卡点。
- [ ] 新伏笔或转折已记录到 `PLOT_POINTS.md`。
- [ ] 新爽点 / 卡点 / 金句已同步回各自的池。

## 特殊章节模板

### 第 1 章（开局章）

```markdown
# 第01章 [章节名]

> **第一句话**：（必须是反常情境 + 被打破的预期）
> **前 300 字名场面**：（必须能直接当短视频素材）
> **冲突钩**：（前 500 字内出现，必填）
> **关系钩**：（核心关系角色登场，必填）
> **反差钩**：（女主反差感的第一处暗示，必填）
> **女主初始困境**：
> **谁在受苦 / 谁在施害 / 为什么关心**：
> **三章内小爽点预告**：（证据留存 / 退路铺好 / 小反击 / 旁人替读者发声）
> **章末卡点**：（必须达到"极强"，推荐人物登场 / 称呼变化 / 信息炸弹）
> **可切片名场面**：
> **金句候选**：（必填，这句要在结局回收）

---

（正文）

---

> **回写标记**：本章需要在大纲完成、第 10 章完成、结局完成后各回写一次。
```

### 第一付费节点章（一般第 3-6 章）

```markdown
# 第XX章 [章节名]

> **付费节点标记**：第一付费节点
> **本章爽点**：（必须是大爽点之一）
> **章末卡点强度**：极强
> **卡点类型**：（推荐 信息炸弹 / 反转预告 / 决定瞬间）
> **不付费就受不了的信息缺口**：（必填，写清读者会想知道什么）
> **高潮前一秒**：（真相/证据/反击种子刚出现，但行动尚未展开）
> **承诺爽感**：（下章必须立刻兑现）
> **禁止提前满足**：（本章不能把哪些答案讲完）

---

（正文）

---

> **付费转化关键点**：本章末必须让读者产生"必须付费看下一章"的冲动。
> **下章兑现要求**：下一章开头必须承接本章卡点，并在 800 字内兑现至少一个爽感。
```
