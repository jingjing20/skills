## 默认补全规则

用户信息不足时，**必须**补齐这些字段：

1. 题材 × 钩子手法（可叠加，两层结构）
2. 篇幅、章节数、免费章数、第一付费节点位置
3. 女主反差感（普通起点 + 隐藏强项）+ 三锚点
4. 男主四件套（专属感 / 反差点 / 暗恋证据链 / 身体吸引力锚点）+ 三锚点
5. 第一句话信息缺口 + 前 300 字名场面 + 第 1 章末男主线索
6. 能进入正文的反击 / 关系张力场景
7. 前 6 章节奏和第一付费节点
8. 2w-5w 字能完成的主冲突
9. 结局方向
10. 真人语感基准、角色口吻差异、AI 腔禁区

## 检查项

提示词完成前必须确认：

- [ ] 题材和钩子手法都已明确（两层结构）
- [ ] 第一付费节点的位置和下一章兑现已确定
- [ ] 第一句话能制造"为什么/然后呢"的信息缺口
- [ ] 前 300 字有具体场景，且完成受害/施害/冲突/关心理由
- [ ] 第 1 章末男主线索已规划（花/卡片/名字/背影/电话）
- [ ] 女主反差感不是抽象描述，可落到具体场景
- [ ] 男主四件套各填齐：专属感 3 条、暗恋证据 3 条、身体吸引力锚点 3 个
- [ ] 男女主三锚点都填齐
- [ ] 反击 / 张力场景具体到动作和代价，不是抽象标签
- [ ] 前 6 章没有连续只虐不反抗
- [ ] 卡点池写清读者不看下一章会缺什么
- [ ] 金句风格示范符合角色口吻，不像广告词
- [ ] 已写清真人语感基准、角色口吻差异和 AI 腔禁区
