# 常见失败模式

用于章节自检、重写和错误记录。发现以下问题时，先修正当前章节，再继续写下一章。

| 类型 | 反例 | 正例 |
| --- | --- | --- |
| 开篇慢热 | “雨很大，城市像一场梦。” | “我替丈夫养了三年的猫，今天才知道猫主人是他的初恋。” |
| 第一章无缺口 | 用 500 字写女主多美多孤独 | 第一段就让女主撞见背叛、羞辱或反常任务 |
| 第一章无男主线索 | 整章都没有男主任何痕迹 | 章末必须出现男主的痕迹（花/卡片/名字/背影/电话） |
| 只虐不爽 | 连续三章女主被压制，没有任何反击 | 第二章偷偷留证，第三章用证据打回去 |
| 爽点密度不足 | 整章只有 1 个爽点 / 全是受虐 | 每章至少 2-3 个爽点（1 主爽 + 1-2 辅助） |
| 爽点三件套不全 | 只有动作没有金句，或没有当众反应 | 必须齐：动作 + 金句 + 对手或旁人当众失态特写 |
| 反派全身而退 | 反派出场后没有当章受挫，下章再算账 | 出场反派必须当章被怼/丢脸/算回去 |
| 反派降智 | 反派愚蠢到全靠主角嘴炮就能赢 | 反派有自己的逻辑链；但被算计、被反将、当场失态都是合法爽点 |
| 付费点过晚 | 真相全讲完再让读者付费 | 女主看到证据第一帧，正要点开完整视频时卡住 |
| 关系跳步 | 上一章还冷脸分手，这章莫名抱在一起 | 中间必有具体事件：陪护一晚、共同危机、旧物触发 |
| 女主失能 | 关键选择都由男主或外力完成 | 主动选择必须由女主触发，男主只能助攻 |
| 误会降智 | 看见男主和女配抱在一起，不问一句就分手 | 看见后冷静离场，第二天派律师送协议，男主才意识到她要走 |
| 情绪断裂 | 上一章还在哭，这章突然嘻嘻哈哈 | 中间必须有情绪过渡场景：独处、回忆、关键对话 |
| 篇幅失控 | 中段反复引入新角色和新支线 | 中段所有新元素必须服务主线，无关支线砍掉 |
| 结局仓促 | 最后一章硬解释所有事 | 最后一章只做情绪落点，外部问题在倒数第 3 章前基本解决 |
| 爽点抽象 | “女主感到很爽” | 具体落到一个动作 / 一句话 / 一个画面 + 对手反应 |
| 卡点疲软 | 章末“她回家睡了一觉” | 章末：“他从背后叫住她，喊出了五年没人叫过的旧名字。” |
| 暧昧跳级 | 第 2 章就写接吻 / 第 3 章就写过夜 | L1-L2 第 1-3 章；L3 第 4-6 章卡付费；L4 第 5-8 章释放；L5 暗示式第 10 章后 |
| 暧昧太克制 | 全程止步 L3，关系永远不升级 | L4 必须从第 5-8 章起释放一次；不能拖到结局 |
| 只氛围不身体 | 写了"暧昧的空气"但没有任何身体反应 | L2 起每个场景至少 2 处具体身体反应（耳尖红、呼吸停、腿软、指尖蜷） |
| 男主没有身体引力锚点 | 男主只有"长得帅"，没有可触发细节 | 必须固定 3 个身体细节锚点（喉结/指节/锁骨/腰线 选 3），全篇反复触发 |
| 软色情场景过界 | 写完整生理过程 / 平台下架风险 | 用清晨钩、痕迹钩、衬衫扣、唇齿——过程留白结果可见 |
| 男主完全失控伤害女主 | 粗暴掐脖、强按、无视女主拒绝 | 克制中短暂失控；失控后必须有反思（"我以为我能控制住自己"） |
| 高级感克制爽文 | 主角"优雅地不计较"、"温柔地放过对方" | 投流读者要直给的爽——破产、入狱、跪求、社会性死亡 |

## 处理规则

- 开篇失败：重写第一句和前 300 字，不要微调几句糊弄过去。
- 付费点失败：后移满足点，前置反击种子，把行动兑现留到付费章。
- 爽点失败：补齐三件套（动作 + 金句 + 当众反应特写）；爽点密度不足直接补 1-2 个。
- 暧昧太克制：补 L4 释放章；男主必须有一次"克制中失控"的破功瞬间。
- 关系跳步：补一个能解释关系变化的事件，不要靠内心戏硬拧。
- 高级感爽文：把"她优雅地"全部改成"她让对方破产"；把"她选择原谅"全部改成"她让对方跪求"。
- 连续性错误：记录到 `ERRORS.md`，同步修正 `STATE.md`、`PLOT_POINTS.md`、`STORY_BIBLE.md`。
