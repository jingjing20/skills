# 女频投流付费核心读点手册

用于投流引导付费阅读场景的女频短中篇。这里写的不是文学规范，是**让读者忍不住付费往下看**的工程规范。

## 一、读者代偿模型

女频投流读者付钱本质上是付"情绪代偿费"。她们需要在故事里完成一次现实里做不到的释放：

| 现实状态 | 故事里要给的代偿 |
| --- | --- |
| 被低估 | 装柔实强 / 身份反转打脸 |
| 不被偏爱 | 男主对女主的专属偏爱 |
| 关系被亏欠 | 男主跪式追妻、对方付出代价 |
| 怕选错人 | 男主用行动反复证明心意 |
| 想被看见 | 男主"读懂"女主的细节描写 |
| 想被托底 | 男主出现的瞬间替女主挡所有事 |

**写作铁律**：每一个爽点都必须能对应上面一种代偿。如果你写出一个"看起来很燃但不让读者代偿"的场景，删掉。

## 二、爽点类型与落点示范

爽点必须**具体落到一个动作、一句话、一个画面**。"她感到很爽"不算爽点。

### 1. 打脸

看低女主的人当场被现实反扑。

正例：

- 婆婆当众说她配不上儿子，男主进门第一句："这房子写她名字。"
- 前夫炫耀新女友身价，下一秒拍卖会上女主举牌报出十倍价格。
- 同事嘲她离婚要不到孩子，律师函上写着孩子的实际监护权一直在她名下。

禁忌：打脸不能靠主角降智把对手设计成弱智。对手必须有自己的逻辑，结果却被女主一记反手破局。

### 2. 反转

身份 / 关系 / 立场的真相揭露。

正例：

- 男主一直被以为是落魄律师，结果是律所幕后合伙人。
- 女主一直被当成普通员工，其实是公司大股东的女儿。
- 五年前男主"出轨"现场，实则是替女主父亲挡刑事责任。

节奏：反转必须早早埋伏笔。前 3 章至少埋 2 个反转伏笔，中段揭一个，结局揭一个。

### 3. 占有欲外显

男主在公开场合明确"她是我的"。重点是公开、被见证、有当事人在场。

正例：

- 酒会上女主被搭讪，男主直接把外套披上去："这是我太太。"
- 公司会议上有人质疑女主任命，男主把任命书甩在桌上。
- 前男友在场，男主当众握住女主的手。

禁忌：不要写成霸道总裁式控制狂。占有欲必须是"对外宣示"，对女主本人是温柔克制的。

### 4. 跪舔追妻 / 火葬场

男主放下身份姿态主动靠近，代价越大越爽。

正例：

- 男主在女主公司楼下等三天，连续被前台请走。
- 男主跨城追到女主新公司，递的不是花，是她父亲案子的翻案证据。
- 男主跪在雨里只说一句话："我错了，但你别走。"

节奏：火葬场段必须放在 65%-85% 的位置，最后 15% 留给和解和落点。

### 5. 装柔实强

女主表面退让实则掌控全局。

正例：

- 婆婆刁难，女主笑着应下，回头一份分家协议甩到桌上：每一笔都合规。
- 前夫提出复合，女主温柔答应见面，见面时桌上摊着他的婚内财产清单。
- 渣女当众挑衅"你能做什么"，女主低头喝茶，三秒后秘书进来递文件："收购完成。"

关键：女主不能靠"嘴炮"赢，必须靠"前期不动声色的布局"赢。布局必须在前面章节铺过，结果在打脸章兑现。

### 6. 旧账清算

过去的伤害被一次性还回去。

正例：

- 五年前抢走女主项目的人，被女主当成跳板买下整个公司。
- 当年逼女主签离婚协议的婆婆，老年痴呆后只认得女主。
- 校园时期欺凌过女主的人，求职面试官就是女主。

禁忌：清算不能是单纯报复。必须有"女主选择了不彻底毁掉对方"或"对方意识到错误"的瞬间，否则爽感会变成戾气。

### 7. 公开偏爱

男主在众人面前为女主破例。

正例：

- 男主从不出席家族饭局，为了女主第一次出席。
- 男主十年不为任何人改行程，为了女主的航班延期。
- 男主对所有人都疏离客气，唯独看见女主时眼神会亮一下（必须是被路人 / 同事注意到的细节）。

### 8. 身份反转

女主真实身份 / 实力被揭露。

正例：

- 一直被当成贫穷打工人的女主，其实是隐姓埋名的钢琴家。
- 一直被婆家以为"配不上"的女主，娘家祖父是当年男主父亲的恩人。
- 一直在公司被欺压的实习生，其实是来收购公司的考察方。

节奏：身份反转一般放在第 1 付费节点（第 3-6 章末）或大爽点章（中段）。

## 三、男主三件套（不可缺一）

### 1. 专属感

男主有一系列"只对女主例外"的行为细节。这是嗑 CP 的核心源代码。

清单要求：至少铺 3 条专属感，每条要在不同章节自然触发，结局必须回收至少 1 条。

示范：

- 对所有人说"您"，唯独对女主说"你"。
- 从不让任何人碰他的左手腕（旧疤），但允许女主无意识握住。
- 抽屉里压着一张被反复折过的便签（女主多年前写的）。
- 永远在女主到公司之前到，从来不让她看见自己等。
- 公司全员加班他不在乎，唯独女主加班到 9 点他会派车。

### 2. 反差点

人前 X / 人后 Y。男主必须有一个稳定的反差结构。

经典反差结构：

- 商场冷血 / 私下笨拙
- 公开温和 / 私下控制（控制只对感情，不对女主本人）
- 表面克制 / 内心剧烈
- 看起来花心 / 实际只对女主上心
- 看起来无情 / 实际记得女主所有细节

禁忌：反差不能是"道德层面的矛盾"。不要写"对所有人渣，对女主好"，那是恋爱脑反派。要写"对所有人专业，对女主破例"。

### 3. 暗恋证据链

男主必须早就在意女主。早期铺设的细节，结局必须由女主或读者意识到。

清单要求：至少 3 条暗恋证据，分布在前 1/3 篇幅，结局回收。

示范：

- 女主多年前提过想去的城市，男主公司在那里开了分公司。
- 女主大学时丢的那本书，他书架上有同一版本，扉页还在。
- 女主以为他不记得自己生日，结果他从不在那一天接重要会议。
- 女主曾在地铁站给乞讨者钱，男主从那天开始每月固定捐同一个站台。

回收方式：不要让男主一次性自爆。让女主自己一点点发现，或由第三方提及。

## 四、女主反差感

女频女主必须有"普通起点 + 隐藏强项"的反差。这是让读者代入的钩子。

### 反差结构清单

| 表面 | 内核 |
| --- | --- |
| 离职律师 | 业内传奇翻案手 |
| 灰头土脸的工地主管 | 海归建筑设计师 |
| 总被欺负的实习生 | 收购方派来的调研员 |
| 看起来恋爱脑 | 实际清账高手 |
| 看起来软弱 | 关键时刻冷得吓人 |

### 落地规则

1. 普通起点必须真实：第 1-3 章必须让读者代入"她也是普通人"。不要一上来就大女主光环。
2. 隐藏强项必须铺线：在被揭露前，至少有 2 处"她其实知道 / 能做什么"的暗示。
3. 强项揭露必须有触发事件：不是炫技，是被逼出来的。
4. 强项揭露后女主不能膨胀：依然保留普通人的疲惫和情绪。

## 五、嗑 CP 三锚点

每个核心角色必须设定三锚点。详细落地规则见 SKILL.md 第四步。

### 视觉锚点

可重复出现、可被截图传播的视觉细节。

好锚点：

- 男主左手腕的浅疤（生气时会下意识捂住）
- 男主深色西装袖口的旧手表（女主多年前送的）
- 女主总是带的同一支墨绿色钢笔

坏锚点：

- "他很帅"——不可视觉化
- "她很温柔"——不可重复触发

### 语言锚点

可被读者记住、可在不同章节复现的语言习惯。

好锚点：

- 男主只在私下叫女主"知知"
- 男主对所有人说"麻烦你"，对女主说"麻烦"（短一个字，亲密差）
- 女主习惯说"我知道了"代替"好"（克制感）

坏锚点：

- "他说话很有磁性"——不可文字化
- "她声音很好听"——不可复用

### 情绪锚点

什么场景必然触发什么反应。这是关系推进的可控变量。

好锚点：

- 雨天，男主必然来接女主（用于章末卡点）
- 女主揉太阳穴，男主必然立刻停止争论
- 男主松领带的瞬间，意味着他真正放下防备（只在女主面前出现）

## 六、读者预期管理（章节级）

读者的耐心阈值是按章计算的。投流读者尤其没耐心。

### 章级期待

| 章节范围 | 必须给读者什么 |
| --- | --- |
| 第 1 章 | 3 重钩 + 必须想点下一章 |
| 第 2-3 章 | 第一个小爽点 + 关系明确化 |
| 第 1 付费节点（3-6 章） | 一个不付费就受不了的强卡点 |
| 第 7-12 章 | 至少 1 个中爽点 + 第一次正面情感冲击 |
| 中段（30%-60%） | 1 个大反转或大爽点 |
| 火葬场段（65%-85%） | 男主代价具体可见 |
| 结局（最后 5 章） | 收束 + 金句回收 + 第1章呼应 |

### 章级雷点

- 章末无事发生：读者直接退出。
- 整章只有内心戏：读者认为剧情没推进。
- 一章塞 3 个反转：读者疲劳。
- 关键场景被叙述代替：读者错过爽感。

铁律：能用场景演的不要用叙述，能用对白演的不要用心理描写。

## 七、女主决不能做的事

1. 决不"哭着原谅"：原谅必须建立在男主付出实际代价之后。
2. 决不"为他放弃事业"：事业必须保住，是关系的反衬而非牺牲品。
3. 决不"为他和家人撕破脸却被牺牲"：女主的家人立场必须可解释。
4. 决不"误会一秒分手"：误会必须经过冷静处理，否则降智。
5. 决不"靠堕胎 / 死孩子 / 失忆"强行虐：套路化死法只服务情绪代偿，必须配合现代节奏改写。

## 八、男主决不能做的事

1. 决不"无理由出现，无理由消失"：行为必须有动机。
2. 决不"用强行肢体接触表达爱"：占有欲必须是公开的、宣示性的，不是私下逼迫。
3. 决不"对前任比对女主更耐心"：前任必须出现在打脸场景，不能抢戏。
4. 决不"一句话能解释却不解释"：男主的沉默必须有具体压力源（家族 / 案件 / 保护对方）。
5. 决不"放下身段后女主立刻原谅"：和解必须分阶段，给读者完整的"付出 → 看见 → 接受"过程。
