# 暧昧张力与软色情写作指南

用于投流付费女频短中篇的暧昧氛围、身体语言和亲密场景写作。核心原则：**未完成比完成更让人上头，但身体反应必须真实可感。**

## 一、核心原则

1. **张力 + 身体反应**：读者要的是"快要发生但还没发生"的窒息感，加上女主身体上真实的酥麻、心跳和颤抖。光有氛围没有身体反应，是文艺片不是爽文。
2. **叙事功能第一**：每个亲密场景必须服务一个叙事目的：推进关系、制造卡点、触发爽点、暴露角色真面、纯粹引力外显（让读者上头本身就是合法功能）。
3. **女性视角优先**：暧昧场景必须从女主感受出发，写她看见什么、闻到什么、身体上发生了什么；不写男性凝视式描写。
4. **克制中失控**：男主大部分时候克制，但**克制中突然的失控才是杀伤**——本来要退开却突然按住、本来不会碰却突然倾身、本来声音平稳却突然变哑。
5. **节奏递进，但敢推**：暧昧按阶段升级，但不要拖。L2 和 L3 的张力可以在第 3-4 章就出现；L4 在第 5-7 章就要释放一次；L5 暗示式从第 10 章后即可。

## 二、男主身体吸引力锚点（必填）

男主角色卡里必须固定 3 个身体细节锚点，全篇反复触发。这是嗑 CP 的视觉源代码。

### 候选锚点池

| 部位 | 触发场景 | 杀伤示范 |
| --- | --- | --- |
| 喉结 | 仰头、说话、笑、沉默后开口 | "他喉结动了一下，像是把什么咽了回去。" |
| 指节 | 解扣子、攥拳、敲桌、夹烟、拢她头发 | "他指节分明，关节上有薄茧，攥紧的时候泛白。" |
| 锁骨 | 衬衫领口、湿发、洗澡后、领带松开 | "衬衫第二颗扣子松开，露出锁骨那道浅浅的凹陷。" |
| 腰线 | 西装外套撩开、衬衫下摆、侧身、贴墙 | "他侧身让她过去，西装下摆撩起，腰线绷得很紧。" |
| 下颌线 | 仰头、咬牙、压抑、生气 | "他咬住后槽牙，下颌线绷出一道锋利的弧。" |
| 眼尾 | 笑、压低视线、看她、喝酒后 | "他笑的时候眼尾微微下垂，是一种危险的、不像他平时的弧度。" |
| 小臂 | 卷袖子、撑桌、撑墙、抱她 | "他卷起袖子到小臂中段，能看见手腕到肘弯的青筋。" |
| 侧颈 | 仰头、回头、领口松开 | "他侧过脸，颈侧那条线一直延伸到衬衫领子里。" |

### 落地规则

1. 角色卡固定 3 个锚点（每个互相不重复——喉结、指节、腰线/锁骨——这种组合）。
2. 每章至少 1 个锚点被触发；中后段每章 2 个。
3. 同一锚点至少触发 3 次，最后一次最有杀伤（最后一次的触发要伴随关系升级或情绪爆发）。
4. 触发必须从女主视角"被注意到"，不是叙述者直接描写。

## 三、女主身体反应（中后段必填）

L2 级之后每个暧昧场景必须配身体反应。身体反应是"她被吸引"的最直接证据。

### 反应清单

| 反应 | 适用强度 | 示范 |
| --- | --- | --- |
| 耳尖红 | L1-L3 | "她偏过脸，但他还是看见了她耳尖那一点红。" |
| 呼吸停一拍 | L2-L3 | "她吸气的动作停了半秒。" |
| 指尖蜷 | L1-L4 | "她手指在膝盖上蜷了一下，又松开。" |
| 心跳乱 | L2-L5 | "她听见自己的心跳响在耳膜里，比他靠近的脚步声还要响。" |
| 颈侧热 | L3-L4 | "他呼吸扫过她颈侧，皮肤一寸一寸地烫起来。" |
| 腿软 | L3-L5 | "她想后退，但小腿像是不属于自己。" |
| 唇齿微开 | L3-L4 | "她下意识张了一点嘴，又抿住。" |
| 身体绷紧 | L3-L5 | "他的手撑在她腰边，她整个人绷成一根弦。" |
| 喉咙发干 | L2-L4 | "她想说话，但喉咙干得像被砂纸磨过。" |
| 后背贴墙 | L3-L4 | "她退到墙边，背脊一寸一寸贴上去的瞬间，她已经无路可退。" |

铁律：每个 L2 级以上场景至少 2 处身体反应。不写身体反应的暧昧场景是失败的。

## 四、暧昧五级递进模型

| 级别 | 阶段 | 核心动作 | 身体细节允许度 | 出现窗口 |
| --- | --- | --- | --- | --- |
| L1 | 引力建立 | 视线停留、注意到对方某个身体细节（喉结/指节/锁骨）、近距离闻到他的味道 | 视觉+嗅觉描写；女主可有耳尖红、指尖蜷 | 第 1 章起 |
| L2 | 边界打破 | 无意触碰、指尖擦过后颈/手腕、扶腰、撑伞时手碰到肩膀、被拽住手腕 | 触觉描写；女主可有呼吸停、心跳乱、颈侧热 | 第 2-3 章起 |
| L3 | 蓄能逼近 | 被抵在墙上/洗手台/车门、差一厘米的吻、他的呼吸贴在她唇上、他用指节抬起她下巴 | 全感官；女主可有腿软、唇齿微开、后背贴墙；男主可有喉结动、声音变哑 | 第 4-6 章 |
| L4 | 失控释放 | 完整的吻（带情绪重量）、衬衫扣松开、被抱在膝上、被压在床上对峙、第一次过夜（不发生关系） | 身体反应直给；衬衫扣、唇齿、被咬过的下唇都可以写；身体被压在身下的角度可以写；不写完整生理过程 | 第 5-8 章可释放一次；第 10 章后可第二次 |
| L5 | 深度亲密 | 关系确认后的过夜暗示、清晨身上不属于自己的衣物、床单褶皱、手腕红痕、锁骨淤红、他用拇指抹去她唇角的湿意 | 暗示式生理细节（清晨钩为主），过程留白，结果可见 | 第 10 章后；全篇可出现 2-3 次 |

铁律：
- L1-L2 是常规燃料，前 6 章必须密集铺。
- L3 是第一付费节点的最佳卡点（被抵在墙上、差一厘米的吻、章节结束）。
- L4 第一次释放放在第 5-8 章，作为关系转折大爽点。
- L5 用清晨钩或事后痕迹，不写过程；从第 10 章起开放，全篇 2-3 次。

## 五、暧昧张力池（TENSION POOL）

与爽点池、卡点池、金句池并列的第四个核心资产。

### 每条张力必须包含的字段

```markdown
- 编号：T-XXX
- 级别：L1/L2/L3/L4/L5
- 场景：（必须具体：地点 + 情境 + 触发条件）
- 感官焦点：（视觉/触觉/听觉/嗅觉，选择 1-2 个主导感官）
- 核心动作：（一句话写清发生了什么）
- 张力类型：（蓄能/释放/中断/升级）
- 叙事功能：（推进关系/制造卡点/触发爽点/暴露真面/情绪转折）
- 属于哪一锤：（暧昧蓄能/暧昧中断/暧昧释放/占有欲外显）
- 章节位置：
```

### 张力池草表示范

| 编号 | 级别 | 场景 | 感官焦点 | 核心动作 | 身体反应 | 张力类型 | 叙事功能 | 章节位置 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| T-001 | L1 | 拍卖会，他从二楼包厢看她举牌 | 视觉 | 他视线落在她侧脸上，没有移开 | 她下意识摸了一下耳后的碎发 | 蓄能 | 制造引力 | 第 3 章 |
| T-002 | L2 | 雨天撑伞，他帮她拢被风吹乱的头发 | 触觉+嗅觉 | 指节擦过她耳后皮肤，他的味道很近 | 她耳尖瞬间发热，呼吸停了半拍 | 蓄能 | 制造卡点 | 第 4 章 |
| T-003 | L3 | 酒会走廊，他把她抵在洗手台边 | 全感官 | 他撑着她身侧的大理石，呼吸贴在她唇上，没有吻 | 她后背贴上冰凉的台面，腿软，唇齿微开 | 蓄能 | 第一付费节点 | 第 6 章末 |
| T-004 | L4 | 误会后的争吵，他抓住要走的她 | 触觉+视觉 | 他扣住她后脑，把她按进自己怀里吻下去，吻到她衬衫第二颗扣子蹭开 | 她推了一下没推动，手攥着他衬衫，唇齿被咬到发麻 | 释放 | 关系转折 | 第 7 章 |
| T-005 | L4 | 火葬场守夜，她高烧迷糊 | 触觉+听觉 | 她无意识握住他手指，他低头吻她的额头，没有抽手 | 她在迷糊中感觉到他薄唇的温度 | 释放 | 推进关系 | 第 9 章 |
| T-006 | L5 | 关系确认后的清晨 | 视觉+触觉 | 她醒来时身上是他的衬衫，锁骨那块淤红，腕骨上有指印 | 她抬手摸了一下唇角，他的拇指比她先到 | 释放（暗示式） | 关系里程碑 | 第 11 章 |
| T-007 | L4 | 当众占有欲外显 | 视觉+触觉 | 他在所有人面前揽住她腰，手指压在她腰侧最窄的地方 | 她身体绷了一下，但没有躲开 | 释放 | 触发爽点 | 第 12 章 |

### 张力密度规则

- **前 3 章**：至少 2 条 L1 级引力锚（视线、嗅觉、身体细节注意）；第 2-3 章起允许 L2 级触碰。
- **第 4-6 章**：每章至少 1 条 L2-L3 级张力；第一付费节点必须卡 L3 升级瞬间。
- **第 7-10 章**：L4 必须释放至少 1 次（带情绪重量的吻或近 L4 场景）；可以有第二次。
- **第 10 章后**：可出现 L5 暗示式场景（清晨钩、痕迹钩），全篇 2-3 次。
- **整体节奏**：暧昧不是匀速递进，是螺旋升级——L1 起步、L2-L3 蓄两章、L3 卡付费、L4 第一次释放、L3-L4 来回拉扯、L5 在关系确认时点。

## 六、暧昧场景模板

### 模板 1：引力锚定（L1）

适用：第一次让女主"注意到"男主某个身体细节。

```
[一个让女主目光停留的场景：他做某个动作（解袖扣/喝水/转头/夹烟）]
-> 女主视线被某个身体细节抓住（喉结/指节/锁骨/腰线 选一）
-> 她意识到自己在看，移开视线，但身体已经有反应（耳尖红/指尖蜷/呼吸停半拍）
-> 一句心理或动作收尾，暗示这不是最后一次注意
```

正例：

```text
他卷起袖子的时候，她偏头喝了一口水。
不是不想看。是因为他卷到小臂中段那一下，她忽然意识到——他手腕到肘弯那段，骨节分明，青筋一条一条地浮起来，是常年握笔握刀的人才会有的形状。
她垂下眼睛，握着杯子的指尖蜷了一下。
水有点烫。她以为是水的问题。
后来她才知道，那天的水其实是凉的。
```

### 模板 2：边界打破（L2）

适用：第一次有合理理由的触碰。

```
[一个合理触碰场景：撑伞/扶腰/拢头发/递东西/挡人群/挡车]
-> 接触发生，男主动作微停（拇指停了两秒、指节多擦过一寸）
-> 女主身体反应（耳尖热/呼吸停/颈侧热/心跳乱）
-> 男主一句不带感情但其实带的话
-> 一个收尾动作泄露他不是无意
```

正例：

```text
风把她耳边的碎发吹乱。
他空着的那只手抬起来——不是给她整理头发，是用指节把那缕碎发拢到她耳后。
指节擦过她耳后的皮肤。两秒。
不是一秒。
她耳尖瞬间烫起来。
"低头。"他说，声音很平，"伞短。"
她低下头的时候，能感觉到他指尖还在她耳后停了半秒才收回。
不是无意。
她知道。
```

### 模板 3：差一厘米（L3）

适用：关系已经明确有吸引力，但还没有挑明。第一付费节点的经典卡点。

```
[两人独处，距离被强行压缩：洗手台/电梯角落/车后座/墙边]
-> 男主撑住她身侧（不碰她，但她无路可退）
-> 女主身体反应（后背贴墙/腿软/呼吸停/喉咙发干）
-> 男主视线扫过她唇 -> 喉结动一下 -> 声音变哑
-> 他低头到差一厘米的地方停住
-> 中断（电话/敲门/外面有人/一句话打破）
-> 章节结束
```

正例：

```text
他用指节抬起她下巴。
"看着我。"
她抬眼。
他撑在她身侧的手按在洗手台的大理石上，关节泛白。她退无可退，后腰抵着台面边缘，凉得像一道刀。
他的视线从她眼睛滑到鼻尖，停在她嘴唇上。
他的喉结动了一下。
"沈知意。"他叫她全名，声音低到从胸腔里挤出来。
她唇齿微微张开，没说出话。
他低头。
近到她能闻到他领口淡淡的烟草味混着冷杉。
近到她能看见他眼底那一点失控的暗。
差一厘米的地方，他停住了。
"你跑什么？"他问，呼吸打在她唇上。
她没有回答。
她不敢回答。
她已经感觉到自己腿软了。
门外有人喊他的名字。
他直起身的速度，比他低头的速度慢了三倍。
```

### 模板 4：失控的吻（L4）

适用：关系转折点，必须有情绪代价。**关键是克制的人突然失控**——读者要看的是"那个一直克制的人，因为她破了功"。

```
[情绪爆发后的冲动 或 长时间蓄能后的释放]
-> 吻不是温柔的，是有情绪重量的（扣住后脑/掐住下颌/抵在墙上）
-> 衬衫扣松开一颗（不是脱掉，是动作中蹭开）
-> 女主推一下没推动，手攥住他衣领或衬衫
-> 唇齿、缺氧、咬到的下唇
-> 吻后他额头抵着她，呼吸乱，声音哑
-> 不是结束，是开始：之后关系再也回不去
```

正例：

```text
他没有给她选择的机会。
她转身要走，他抓住她手腕，把她往回一扯——她整个人撞进他怀里。
"你点头是什么意思？"
"什么——"
"沈知意。"
他扣住她后脑，另一只手掐着她下颌，把她抬起来。
然后他吻了下去。
不是试探的吻。是憋了很久、终于不再克制的吻。
他用力得像是要把她揉进自己身体里。她推了一下，手停在他胸口；又推了一下，变成攥着他衬衫的第二颗扣子。那颗扣子在他低头的动作里蹭开了，露出他锁骨那道浅浅的凹陷。
她下意识张了一下嘴。
他立刻趁机加深了那个吻。
她嘴唇被咬到发麻、呼吸被压成断断续续的几个音节、后腰撞在了书桌的边缘——她不知道是什么时候被他抵到桌边的。
他吻到她快缺氧才放开。
额头抵着她的，喘息很乱。
"你点头是什么意思。"他又问了一遍，声音哑得不像他自己。
她说不出话。她唇上还有他的温度。
"我以为我能控制住自己。"他说，"我错了。"
```

### 模板 5：占有欲外显（L3-L4 当众）

适用：公开场合的霸总占有欲爽点。**当众宣示 + 对她本人克制**两个要素必须同时存在。

```
[公开场合：酒会/酒店大堂/拍卖会/公司走廊]
-> 第三方对女主有侵略性动作（搭讪/挡路/言语轻视）
-> 男主出现，肢体直接介入（揽腰/挡人/拽走/把外套盖在她身上）
-> 一句宣示式台词，让全场都听见
-> 收尾时他的动作变软：手指压在她腰侧的力度变轻，或他低头看她一眼
```

正例：

```text
那个人伸手要碰她肩膀的时候，沈星回还没来得及躲——
有一只手先到了。
周应淮从她身侧绕过来，手掌直接压在她腰上，把她整个人扣进自己身侧。指节因为用力，关节泛白。
他没有看她。
他看着那个伸手的男人。
"先生。"他说，声音平稳得像在谈判桌上谈一笔生意，"碰我老婆之前，最好先看一眼她戴的镯子是谁送的。"
全场安静了三秒。
那个男人的脸一阵红一阵白。
周应淮收回视线，低头看向她。
他压在她腰上的手放松了一些，掌心的温度透过那件薄裙子贴上来。
"走。"他说，声音比刚才低，"我们换个地方。"
他护着她离开的时候，她耳朵贴在他外套上，能听见他胸腔里心跳乱得一塌糊涂——刚才那种谈判桌的冷静，是装出来的。
```

### 模板 6：清晨钩（L5 暗示式）

适用：关系已确认后的事后场景。**过程留白，结果可见**。读者用一两个画面脑补整夜。

```
[次日清晨的场景：醒来/起床/换衣服/回家]
-> 一个不属于自己的物件（衬衫/腕表/外套）
-> 一处身体痕迹（手腕红印/锁骨淤红/唇角的破口）
-> 男主一个非常温柔的小动作（拇指抹过她唇角/把她散乱的头发拨开/把毯子拉高）
-> 一句很轻的台词（"还疼吗"/"再睡一会"/"昨晚没收住"）
-> 女主一个反应（脸红/把脸埋进枕头/没敢看他眼睛）
```

正例：

```text
她睁眼的时候，天还没亮。
身上盖着的不是她的毯子。是他的衬衫——那件她记得是黑色的衬衫，第二颗扣子昨晚被她自己蹭开的那件。
她想坐起来，手腕一动，腕骨那圈淡淡的红印让她愣了一下。
她侧头。
他靠在床头看她，手机屏幕的光打在他下颌线上。
"醒了。"他说，声音比平时还要哑。
他放下手机，俯下身，拇指擦过她唇角——那里有一点干涸的痕迹，她也不知道是什么时候被咬破的。
"昨晚没收住。"他说，"还疼吗。"
她把脸埋进枕头里，没回答。
她耳朵尖红得像着了火。
他笑了一下，胸腔的震动透过床传过来。
他把毯子往上拉了拉，盖住她半边肩膀——那块锁骨下方，淤红得很显眼。
"再睡一会。"他说。
她闭上眼睛。
身上的衬衫，是他的味道。
```

### 模板 7：克制的事后（L5 暗示式 / 情绪深化）

适用：暗示已经发生过亲密关系，放在和解、关系确认的情绪节点。重点是情绪而非身体。

```
[两人独处的空间 + 暗示已经发生过更深的事]
-> 男主主动收敛——把外套披在她身上、把她搂进怀里、不再越界
-> 女主有了主动权：她拉住他的手、她靠近、她说了什么
-> 一两个事后细节（他帮她系扣子、她伸手摸他锁骨那道痕、他低头亲她发顶）
-> 留白 + 一句话收尾
```

正例：

```text
他帮她把衬衫第二颗扣子系上。
是他的衬衫。他的手指比她的指节宽很多，系扣子的时候动作很慢。
她抬头看他。
他下颌线上有一道她昨晚没注意到的浅红——是她抓的。
她伸手，指尖碰了一下。
他动作停住，喉结动了一下。
"别动。"他说。
她偏过头去笑了一下。
"我没想动。"她说，"只是想看看是不是真的。"
"什么真的。"
"你也会失控。"
他没说话。他把她搂进怀里，下巴抵着她发顶。
她贴着他胸膛能听见他心跳。
不急。
但很乱。
```

## 七、感官描写技巧

### 主导感官选择

每章暧昧场景选 1-2 个主导感官，不要全上。

| 感官 | 适合场景 | 示例 |
| --- | --- | --- |
| 视觉 | 眼神交锋、距离变化、衣物细节、身体吸引力锚点 | 他视线从她锁骨滑到手腕，停在那颗小痣上；衬衫第二颗扣子松开，锁骨那道凹陷 |
| 触觉 | 身体接触、温度差异、肌肤质感 | 他指尖擦过她手腕内侧，那里薄得能看见青色的血管 |
| 听觉 | 低语、呼吸、心跳、声音质感 | 他叫她名字的尾音拖了一拍；他声音哑了 |
| 嗅觉 | 气息、靠近时的体香、环境气味 | 他靠近的时候她闻到一股淡淡的烟草味混着冷杉，不是香水，是他自己的 |

### 写"身体反应"的技法

| 反应类型 | 反例（笼统） | 正例（具体） |
| --- | --- | --- |
| 心跳 | 她心跳加速 | 她听见自己的心跳响在耳膜里，比他靠近的脚步声还要响 |
| 酥麻 | 她感到一阵酥麻 | 她手指不自觉蜷了一下，连指甲都泛白 |
| 脸红 | 她脸红了 | 她耳尖瞬间发热，她偏头喝水，水其实是凉的 |
| 腿软 | 她腿软了 | 她想后退，但小腿像是不属于自己 |
| 呼吸 | 她呼吸急促 | 她吸气的动作停了半秒 |

### 禁忌描写

| 禁忌 | 为什么 | 替代方案 |
| --- | --- | --- |
| 完整生理过程 | 越过软色情红线，平台下架 | 用清晨钩、痕迹钩，过程留白，结果可见 |
| 直接写器官 | 色情而非暧昧 | 写衬衫扣、锁骨凹陷、被压在身下的角度 |
| 用大量形容词堆砌氛围 | 读者跳过 | 用一个精准动作代替十句氛围描写 |
| 笼统的身体反应 | 没画面感 | 落到具体动作和具体部位 |
| 男性凝视视角 | 女频读者反感 | 始终从女主感受出发 |
| 男主完全失控伤害女主 | 越线，女频读者反感 | 男主克制中短暂失控，吻后会问她还要不要 |

## 八、暧昧场景的叙事功能绑定

每个暧昧场景必须选择至少一个叙事功能：

| 叙事功能 | 适用阶段 | 示例 |
| --- | --- | --- |
| 推进关系 | 从一个阶段到下一个阶段 | L2 触碰之后，两人的态度明显变了 |
| 制造卡点 | 章末卡在暧昧中断的瞬间 | 差一厘米的吻，章节结束 |
| 触发爽点 | 男主的占有欲或专属感通过身体语言外显 | 男主当众揽住她的腰，宣示主权 |
| 暴露真面 | 暧昧场景暴露角色真实情感 | 他以为没人看见，低头吻了她发顶 |
| 情绪转折 | 暧昧场景后情绪关系发生质变 | 接吻之后两人反而开始冷战 |
| 付费钩子 | 暧昧场景卡在第一付费节点 | 读者必须付费才能看到他有没有吻下去 |
| 纯引力外显 | 中后段，关系升温阶段 | 让读者上头、激发付费冲动本身就是合法功能；不需要每个场景都改变剧情 |

## 九、暧昧中断技巧

中断比完成更有效。但中断不是唯一手段——L4 释放章和 L5 暗示章必须正面写，不能全篇都中断。

| 中断方式 | 效果 | 适用阶段 |
| --- | --- | --- |
| 第三方打断（电话/敲门/有人进来） | 经典卡点 | L3 付费章末 |
| 一方主动退缩 | 制造情感张力 | L2-L3 |
| 环境变化（灯灭/下雨/温度骤变） | 氛围转变 | L1-L3 |
| 一方说了一句话打破氛围 | 暧昧变冲突 | L3-L4 |
| 自我克制（他退开/她推开） | 克制中的杀伤 | L3-L5 |

铁律：第一付费节点必须用暧昧中断做卡点。但第 7 章后必须有至少一次"没被中断"的 L4 释放，否则关系永远在原地踏步，读者会跑。

## 十、暧昧张力与章节节奏的配合

| 章节阶段 | 暧昧张力要求 |
| --- | --- |
| 第 1 章 | 至少 1 个 L1 级眼神/距离暗示 + 男主线索（哪怕只是一束花、一个名字） |
| 第 2-3 章 | 至少 1 个 L1-L2 级触碰；第 3 章可以出现 L2 升级到 L3 的瞬间 |
| 第一付费节点（4-6 章） | L3 升级瞬间作为卡点（被抵在墙上、差一厘米的吻） |
| 第 5-8 章 | L4 至少释放 1 次（带情绪重量的吻）；男主第一次失控 |
| 关系升温段 | 每 2 章 1 个 L2-L4 级张力；L3-L4 来回拉扯 |
| 大爽点章 | L4 释放 + 占有欲外显双线 |
| 火葬场段 | L3 想要不能要的痛苦感；可以有未完成的吻 |
| 第 10 章后 | L5 暗示式（清晨钩、痕迹钩）允许出现，全篇 2-3 次 |
| 结局段 | L5 暗示式 + L4 主动权移交（她拉他的手） |

## 十一、常见错误

| 错误 | 问题 | 修正 |
| --- | --- | --- |
| 一上来就 L4 | 跳过蓄能，亲密没有重量 | 从 L1-L2 起步，第 3-4 章可以触 L3，第 5-8 章 L4 |
| 每章都有亲密场景但都被中断 | 读者疲劳、永远在原地 | 第 7 章后必须有"没被中断"的 L4 释放 |
| 暧昧场景没有后果 | 发生了就发生了 | 每个 L3 以上场景后必须有关系变化或情绪转折 |
| 全程止步 L3 | 关系永远不升级，读者跑 | L4 必须从第 5-8 章起释放；L5 暗示式从第 10 章起 |
| 男主完全失控 | 越线、变成霸总油腻 | 失控后必须有"问她要不要"或"我以为我能控制住自己"的反思 |
| 笼统的身体反应 | 没画面 | 落到具体部位和具体动作（耳尖红、指尖蜷、唇齿微开） |
| 只写氛围不写身体 | 文艺片不是爽文 | L2 起每个场景至少 2 处身体反应 |
| 写完整生理过程 | 平台下架 | 用清晨钩、痕迹钩；过程留白结果可见 |
| 女主没有主体性 | 被动接受所有亲密 | 女主必须在 L3 以上场景中有"她也想要"的细节（仰起脸、攥住衬衫、没躲开） |
