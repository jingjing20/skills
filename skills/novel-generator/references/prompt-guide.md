# 女频投流付费短中篇提示词完善指南

从用户的一句话方向，扩展为可投流的女频付费短中篇设定。投流付费场景下，提示词最大的产出物不是"故事梗概"，而是**爽点池 / 卡点池 / 金句池 / 张力池 + 三锚点角色档案**。

## 补全优先级

```text
必须补全：
  1. 题材 × 钩子手法（可叠加）
  2. 篇幅、章节数、免费章数、第一付费节点位置
  3. 第一句话信息缺口
  4. 前 300 字名场面
  5. 女主反差感（普通起点 + 隐藏强项）
  6. 男主三件套（专属感 / 反差点 / 暗恋证据链）
  7. 主冲突和结局方向
  8. 爽点池首批 8 条
  9. 暧昧张力池首批 5 条（L1-L5 逐级分布，含场景+感官+叙事功能）
  10. 前 6 章连环锤节奏
  11. 卡点池首批 6 条，写清读者不看下一章会缺什么
  12. 金句风格示范 3 句

应该补全：
  13. 关键转折分布到具体章节
  14. 误会/秘密的合理来源
  15. 甜虐比例及具体落到哪些章节

可以补全：
  16. 次要角色
  17. 特殊场景
  18. 情绪曲线细节
```

## 题材 × 钩子手法（两层结构）

题材和钩子手法是**可以叠加**的两个变量。古言 + 替嫁、现言 + 重生、仙侠 + 破镜重圆都是合法组合。

### 题材清单

| 题材 | 必须想清楚 |
| --- | --- |
| 现言 | 职业压力、现实阻力、关系选择 |
| 古言 | 身份差、礼法约束、权力风险 |
| 仙侠 | 宿命、牺牲、修行规则如何影响感情 |
| 校园 | 成长、暗恋、分别、重逢或遗憾 |
| 娱乐圈 | 舆论、事业、曝光风险、公众形象 |
| 豪门 | 阶层差、利益交换、家族压力 |
| 年代 | 时代限制、家庭重负、命运挤压 |
| 悬疑爱情 | 案件线如何揭开人物秘密 |

### 钩子手法清单

| 钩子手法 | 关键问题 |
| --- | --- |
| 破镜重圆 | 分手原因必须具体，复合代价必须可见 |
| 追妻火葬场 | 男主代价 / 跪点必须落到具体场景 |
| 久别重逢 | 重逢瞬间必须有"称呼变化"或"物件触发" |
| 先婚后爱 | 婚姻是压力容器，关系升温靠共同经历 |
| 暗恋成真 | 暗恋证据链必须早早埋好 |
| 契约 / 替嫁 | 契约条款是冲突源，期限是节奏推手 |
| 重生归来 | 重生第一击 + 清算节奏 |
| 穿书改命 | 原剧情压力 + 女主自由意志的对抗 |
| 马甲 / 隐藏身份 | 至少 2-3 张马甲，分层揭露 |
| 强强 | 双方能力对等，冲突来自立场 |

## 爽点池设计指南

提示词阶段必须给出**首批 8 条爽点**，写大纲时扩充到 15+。

### 每条爽点必须包含的字段

```markdown
- 编号：
- 类型：（打脸 / 反转 / 占有欲外显 / 跪舔追妻 / 装柔实强 / 旧账清算 / 公开偏爱 / 身份反转）
- 触发场景：（必须具体，地点+在场人物+起因）
- 谁打谁脸 / 谁付出 / 谁意识到：
- 读者代偿点：（对应哪种现实情绪释放）
- 大致章节位置：
- 是否依赖前文铺垫：（是 → 在哪一章铺）
- 属于哪一锤：（受虐 / 小反抗 / 打脸 / 升级反击）
```

### 爽点池草表示范

| 编号 | 类型 | 触发场景 | 谁打谁脸 | 读者代偿点 | 章节位置 |
| --- | --- | --- | --- | --- | --- |
| P-001 | 打脸 | 婆婆在家宴上说女主配不上 | 男主当众宣布房产改写女主名字 | "被低估的人被托底" | 第 4 章末（第一付费节点） |
| P-002 | 反转 | 同事嘲女主只是普通秘书 | 项目签约现场显示女主是甲方代表 | "被看轻者打回去" | 第 7 章 |
| P-003 | 占有欲外显 | 酒会有人搭讪女主 | 男主直接披外套："这是我太太" | "公开宣示的被珍视" | 第 10 章 |
| P-004 | 暗恋证据 | 女主翻男主书房 | 看见自己丢了多年的书，扉页是自己的字 | "被偷偷在意" | 第 12 章 |
| ... |  |  |  |  |  |

铁律：**爽点必须具体到一个动作 / 一句话 / 一个画面**。不接受"女主出了一口气"这种抽象描述。

## 连环锤设计指南

前 6 章必须单独规划连环锤，避免只虐不爽。

```markdown
| 章节 | 施压/受虐 | 女主反应/小反抗 | 本章补偿爽点 | 章末升级 |
| --- | --- | --- | --- | --- |
```

规则：

- 第 1 章可以重锤，但第 2-3 章必须出现小反抗、留证、退路或旁人替读者发声。
- 连虐两章后，下一章必须给补偿爽点。
- 爽点不一定是大反击，但必须让读者相信女主不会一直惨。
- 男主可以助攻，但不能替女主完成关键选择。

## 卡点池设计指南

提示词阶段必须给出**首批 6 条章末卡点构思**。

### 每条卡点必须包含的字段

```markdown
- 编号：
- 类型：（人物登场 / 称呼变化 / 信息炸弹 / 动作未完 / 反转预告 / 误会触发 / 旧物出现 / 决定瞬间）
- 落点画面：（一句话写清章末画面）
- 强度：极强 / 强 / 中
- 读者不看下一章会缺什么：
- 适用章节位置：
```

### 卡点池草表示范

| 编号 | 类型 | 落点画面 | 强度 | 读者不看下一章会缺什么 | 章节位置 |
| --- | --- | --- | --- | --- | --- |
| H-001 | 人物登场 | 推开门，对面是五年没见的他 | 极强 | 他为什么在这里 | 第 1 章末 |
| H-002 | 称呼变化 | 他叫出五年没人叫过的"知知" | 极强 | 他为什么知道旧称呼 | 第 3 章末 |
| H-003 | 信息炸弹 | 鉴定书签字栏是男主的名字 | 极强 | 他当年到底做了什么 | 第 6 章末（第一付费节点） |
| H-004 | 动作未完 | 他抬手停在她耳边一厘米的地方 | 中 | 他敢不敢碰她 | 第 9 章末 |
| H-005 | 误会触发 | 走廊尽头他正抱着另一个女人 | 强 | 这次是不是又背叛 | 第 12 章末 |
| H-006 | 旧物出现 | 抽屉里压着她多年前的便签 | 强 | 他藏了多少年 | 第 15 章末 |

## 金句池设计指南

金句是**可被截图传播的句子**。投流场景下，金句直接作为短视频物料和落地页文案。

### 金句风格示范（3 句）

提示词阶段必须给出 3 句目标语感的金句，作为后续章节产出的语言锚点。

示范：

```text
1. "他从来不擅长挽留。所以这一次，他用了五年的时间，绕回到她面前。"

2. "她说，我不爱你了。
    他说，那我可以等。
    她说，可你不会等的。
    他说，沈知意，你最不了解的人，从来都是我。"

3. "雨下到傍晚还没停。
    像他们之间所有的话——下不完，也说不完。"
```

### 金句产出规则

- 每章至少产出 1 句金句候选，记录到 `GOLDEN-LINES.md`。
- 金句必须有人物名 / 称呼 / 关系信息嵌入，可独立传播。
- 至少 1/3 的金句出现在章末（与卡点结合）。
- 至少 1 句金句在结局回收（与第 1 章呼应）。

## 暧昧张力池设计指南

提示词阶段必须给出**首批 5 条暧昧张力构思**。张力池与爽点池、卡点池、金句池并列，是第四个核心资产。详细写作指南见 `chemistry-guide.md`。

### 每条张力必须包含的字段

```markdown
- 编号：T-XXX
- 级别：L1/L2/L3/L4/L5
- 场景：（必须具体：地点 + 情境 + 触发条件）
- 感官焦点：（视觉/触觉/听觉/嗅觉，选择 1-2 个主导感官）
- 核心动作：（一句话写清发生了什么）
- 张力类型：（蓄能/释放/中断/升级）
- 叙事功能：（推进关系/制造卡点/触发爽点/暴露真面/情绪转折/付费钩子）
- 属于哪一锤：（暧昧蓄能/暧昧中断/暧昧释放/占有欲外显）
- 章节位置：
```

### 暧昧五级递进模型

| 级别 | 阶段 | 核心动作 |
| --- | --- | --- |
| L1 | 眼神与距离 | 视线停留、无意靠近、呼吸可闻的距离 |
| L2 | 无意触碰 | 手指擦过、扶腰、帮整理衣领、碰到手 |
| L3 | 刻意靠近 | 独处、近距离对视、气息交融、差一厘米的吻 |
| L4 | 身体亲密 | 接吻、拥抱、肌肤接触、共枕 |
| L5 | 深度亲密 | 同床、更进一步的身体接触 |

### 张力池草表示范

| 编号 | 级别 | 场景 | 感官焦点 | 核心动作 | 张力类型 | 叙事功能 | 章节位置 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| T-001 | L2 | 雨天车内，男主帮女主擦脸上的水 | 触觉+嗅觉 | 拇指擦过她颧骨，停了两秒才收回 | 蓄能 | 推进关系 | 第 5 章 |
| T-002 | L3 | 酒会走廊，男主把女主抵在墙边 | 视觉+触觉 | 他低头，呼吸扫过她嘴唇，却没有吻下来 | 蓄能 | 制造卡点 | 第 8 章末 |
| T-003 | L3 | 病房，女主发烧说胡话，男主守夜 | 触觉+听觉 | 她无意识握住他的手指，他整夜没有抽走 | 释放 | 推进关系 | 第 10 章 |
| T-004 | L4 | 男主家，争吵后的沉默 | 触觉+视觉 | 他捧住她的脸，拇指按在她唇角，吻了下去 | 释放 | 关系转折 | 第 14 章 |
| T-005 | L5 | 确认关系后的第一个夜晚 | 触觉+氛围 | 他把她拉进怀里，下巴抵着她发顶，低声说不急 | 释放 | 关系里程碑 | 第 18 章 |

### 设计铁律

- 暧昧必须从 L1 逐级递进，不能跳级。
- L4/L5 不能在前 3 章出现。第一付费节点适合卡在 L2→L3 升级瞬间。
- L5 全篇不超过 2 次，且只在关系确认后使用。
- 每个暧昧场景必须有叙事功能，不能为擦而擦。
- 中断比完成更有效：被电话打断的吻、差一厘米的距离、松手太慢的触碰。
- 女主在亲密场景中必须有选择权和主动权。

## 三锚点角色档案

每个核心角色必须设定三锚点。详细规则见 SKILL.md 第四步和 `female-readership.md`。

### 模板

```markdown
## 角色名

**视觉锚点**（至少 2 条，可重复触发）：
- 

**语言锚点**（至少 2 条，可文字化复用）：
- 

**情绪锚点**（至少 2 条，特定场景触发特定反应）：
- 

**当前状态 / 当前位置 / 当前目标 / 内在伤口 / 关系阶段**：
```

铁律：**不写满三锚点的角色不允许参与章节生成**。

## 常见提示词问题修正

### 女主没有主动性

修正：

- 给女主一个必须亲自完成的目标。
- 关键转折必须由女主选择触发。
- 男主可以帮忙，但不能替女主完成成长。
- 女主必须有"被低估 → 反扑"的爽点节点。

### 关系进展太快

修正：

- 每次升温前放一个具体事件。
- 从对立、试探、信任、依赖、选择逐级推进。
- 不要用"忽然心动"替代因果。
- 至少经过 2 次小爽点 + 1 次小虐点才能进入下一阶段。

### 误会太蠢

修正：

- 误会必须来自信息差、身份限制、利益冲突或保护动机。
- 如果一句话能解释清楚，就不要拖成主线。
- 误会解除后必须产生新的选择，而不是故事直接停住。
- 主线误会不能拖超过 3-5 章。

### 篇幅膨胀

修正：

- 只保留一个主冲突。
- 次要角色只服务主线。
- 中后段不再引入新的大地图或大反派。
- 最后 5 章不开新副线。

### 男主没记忆点

修正：

- 三件套必须填齐（专属感 + 反差点 + 暗恋证据链）。
- 视觉锚点和语言锚点必须早早建立。
- 占有欲必须公开、可见、被见证。
- 至少 2 次"人前 X / 人后 Y"的反差场景。

### 没有爽点

修正：

- 爽点池必须达到 15 条以上。
- 每章至少 1 个小爽点。
- 第 1 付费节点必须是大爽点之一。
- 检查每条爽点是否对应一种读者代偿。

### 缺少暧昧张力

修正：

- 张力池必须在提示词阶段给出首批 5 条。
- 前 6 章必须有至少 3 个 L1-L2 级身体吸引力场景。
- 第一付费节点适合卡在 L2-L3 暧昧中断（差一厘米的吻、被中断的触碰）。
- L4/L5 不能在前 3 章出现，必须从 L1 逐级递进。
- 每个暧昧场景必须有叙事功能：推进关系、制造卡点、触发爽点或暴露真面。
- 男主在亲密场景中的克制比冲动更有杀伤力。

## 用户方向解析示例

### 模糊方向

用户输入：

```text
写个现言女频
```

补全思路：

- 默认 3 万字，12-16 章，单章 2000-3000 字。
- 钩子手法：追妻火葬场 + 双重背叛 + 绝地反击（最容易做投流爆款）。
- 设定女主职业和情感伤口（律师 / 设计师 / 医生）。
- 第一句必须直接给信息缺口，前 300 字必须出现具体冲突。
- 设计一个能在 12-16 章内解决的现实阻力（旧案 / 旧债 / 家族压力）。
- 爽点池立刻给出 8 条候选。
- 张力池立刻给出 5 条候选（覆盖 L1-L3，含具体场景和感官焦点）。
- 前 6 章立刻给出连环锤节奏。
- 卡点池立刻给出 6 条候选。

### 半明确方向

用户输入：

```text
破镜重圆，女主是律师，男主是前男友
```

补全思路：

- 明确分手原因（必须是男主"无法解释"的保护性沉默）。
- 明确重逢事件（共同案件 / 共同当事人 / 共同朋友）。
- 明确女主这次不愿回头的真实原因（不只是赌气，还有现实代价）。
- 设计男主需要付出的代价：跨城追、放下身份、揭旧伤、当众认错。
- 爽点池围绕"打脸前未婚妻 + 男主公开偏爱 + 旧案真相 + 男主跪式追妻"展开。

### 详细方向

用户输入：

```text
3 万字，古言，女主替嫁，男主是病弱王爷，先婚后爱
```

补全思路：

- 控制在 18-24 章。
- 明确替嫁风险（家族债 / 替罪羊身份）。
- 明确王府权力关系（嫡庶 / 太后 / 王爷的政敌）。
- 男主病弱真假：必须是装病（爽点反转预留）。
- 情感线从合作到信任，再到共同承担后果。
- 爽点池围绕"装病揭穿 + 王妃身份反转 + 太后被打脸 + 政敌清算"展开。
- 结局收束婚姻、家族、女主身份三个问题。
