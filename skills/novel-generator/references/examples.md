# 示例集

女频短中篇小说的提示词、章节和状态文件示例。

## 示例一：创作提示词

```markdown
# 《春风不渡》创作提示词

## 基本定位
- **类型**: 现言，破镜重圆
- **目标读者**: 女频读者
- **总字数**: 约 30000 字
- **预计章节数**: 12-16 章
- **单章字数**: 2000-3000 字
- **结局**: HE

## 核心卖点
- **一句话梗概**: 离职律师沈知意在一场遗产案中重逢前男友陆沉舟，被迫揭开五年前分手真相，也重新选择自己的人生。
- **第一句话信息缺口**: 沈知意接下离职前最后一单，委托地址却是她五年前逃离的婚房。
- **前 300 字名场面**: 她推开婚房门，看见猫碗旁压着前男友的袖扣和一份写着她名字的遗嘱。
- **情绪关键词**: 克制、酸涩、拉扯、治愈
- **关系关键词**: 破镜重圆、久别重逢、旧误会
- **最大读点**: 两个还爱着的人，被现实和旧伤逼到必须说真话。

## 女主
- **姓名**: 沈知意
- **初始身份**: 离职律师，准备离开本城
- **外在目标**: 完成最后一桩遗产案，攒够钱去南方开小律所
- **内在伤口**: 五年前以为陆沉舟为了前途放弃她
- **成长弧**: 从逃避旧伤到主动追问真相，并重新掌控人生选择

## 男主
- **姓名**: 陆沉舟
- **初始身份**: 商业律所合伙人
- **与女主关系**: 前男友
- **吸引力**: 克制可靠，始终记得女主的理想
- **阻力**: 五年前隐瞒了替女主承担风险的真相

## 主要冲突
- **外部冲突**: 遗产案牵出五年前女主父亲案子的旧证据
- **情感冲突**: 女主不相信陆沉舟，陆沉舟不敢解释过去
- **误会/秘密**: 当年分手不是背叛，而是保护，但保护本身也是伤害
- **必须解决的问题**: 女主是否继续逃离，二人是否能平等地重新开始
```

## 示例二：章节片段

```markdown
# 第01章 她没想到会再见他

> **本章概要**：沈知意接下离职前最后一桩遗产案，在委托人家中重逢陆沉舟。
> **情绪功能**：旧伤重启
> **关系进展**：从五年未见到被迫同案
> **章末钩子**：陆沉舟叫出她五年前只告诉过他的旧称呼

---

雨下到傍晚还没停。

沈知意把辞职信压在办公桌左上角，纸页被空调风吹得轻轻一颤。她看了两秒，还是伸手拿过来，塞进了包里。

主任在电话里劝她：“最后一桩案子，做完再走。委托人点名要你。”

她笑了笑：“我在这家所三年，第一次听说有人点名要我。”

“遗产纠纷，不复杂。麻烦的是对方请了盛景的人。”

沈知意拉拉链的手顿住。

盛景律所。

这个名字像一根很细的针，扎进她早就结痂的地方。她没有问是谁，只说：“地址发我。”

半小时后，她站在城南一栋老洋房门前。雨水顺着伞骨滴落，门铃响过第二声，门从里面打开。

男人穿着深灰色西装，袖口挽起半截，露出腕骨上那道浅淡的旧疤。

沈知意握着伞柄的手指慢慢收紧。

五年不见，陆沉舟还是那副冷静模样。只是眉眼比从前更沉，像把所有话都压进了眼底。

他看着她，先开了口。

“知知。”

她已经很多年没听过这个称呼。

---

> **本章更新**：沈知意与陆沉舟重逢，被迫进入同一桩遗产案。
> **下章目标**：明确案件利益冲突，制造第一次正面对话。
```

## 示例三：STATE.md

```markdown
# 创作状态

## 当前进度

- 已完成字数：约1400
- 目标总字数：30000
- 预计总章节：20
- 剩余章节：19

## 上一章结尾
陆沉舟叫出“知知”，触发沈知意旧伤。

## 下一章目标
- 核心事件：说明遗产案双方立场，女主发现必须与男主合作。
- 情绪推进：尴尬、克制、压抑怒意。
- 关系变化：从逃避到被迫对话。
- 必须避免：不要让两人立刻解释清楚五年前分手原因。

## 待处理事项
- 未回收伏笔：陆沉舟手腕旧疤、五年前分手真相。
- 未解决误会：女主以为男主主动放弃她。
- 必须出场角色：委托人许老太。
```
